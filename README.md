Chrome-расширение, которое перехватывает бэкап [unidraw.io](https://unidraw.io) и конвертирует его в [Mermaid](https://mermaid.js.org/)-диаграмму прямо в браузере.

---

## Как пользоваться

1. Установите расширение (см. ниже)
2. Откройте борд на [unidraw.io](https://unidraw.io/app/board/)
3. Нажмите **"Сохранить бэкап на компьютер"** в меню unidraw
4. Кликните на иконку расширения — диаграмма уже готова
5. Переключайте направление графа (TD / LR / BT / RL)
6. Копируйте Mermaid-код или скачивайте исходный `.unidraw` JSON

Расширение хранит до 20 последних снимков с навигацией по истории и может блокировать отправку бэкапа на сервер (переключатель **"Не сохранять на сервер"** в шапке).

| Вкладка | Что внутри |
|---------|-----------|
| **Диаграмма** | SVG-визуализация Mermaid-графа с выбором направления |
| **Код** | Mermaid-код (копирование) + скачивание исходного JSON |
| **Элементы** | Статистика, список узлов и связей |

## Почему Боб не смог распарсить мою борду?

Боб умеет конвертировать только **фигуры** (shape) и **связи** (arrow, connector, line). Всё остальное — мусор, с точки зрения Боба.

Если на борде встретились:

| Элемент | Что делает Боб |
|---------|---------------|
| Стикеры, текстовые блоки | Выкидывает. Это не узлы графа, это ваши заметки. |
| Картинки | Выкидывает. Боб не умеет читать ваши скриншоты, и не хочет. |
| Freedraw (рисунки от руки) | Выкидывает. Серьёзно, вы рисовали на борде пальцем? |
| Фигуры без единой связи | Выкидывает. Если узел ни к чему не подключён — это не диаграмма, это одиночество. |

Если после фильтрации на борде осталось **0 связей** — Боб радуется и уходит на перерыв. Диаграмму вы не получите, но получите уважение Боба.

Если связи есть, но борда содержит несколько изолированных групп — Боб аккуратно разделит их на **отдельные подграфы**. Не благодарите.

## Установка

### Google Chrome / Opera / Brave / Vivaldi / Яндекс Браузер

1. Скачайте ZIP-файл из [Releases](../../releases)
2. Распакуйте в любую папку
3. Откройте `chrome://extensions/`
4. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
5. Нажмите **"Загрузить распакованное расширение"**
6. Выберите распакованную папку
7. Готово! Иконка расширения появится на панели Chrome

### Microsoft Edge

Наши соболезнования.

### Firefox / Safari

Пока нет. Но Боб уже знает, где вы живёте. Возможно, однажды он доберётся и до вас.
